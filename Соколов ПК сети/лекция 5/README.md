### Конспект лекции (для отчета)

**Введение в Информационную Безопасность (ИБ)**

- **ИБ** — это область знаний и методов, направленных на защиту информации от кражи, изменения, уничтожения и несанкционированного доступа.

- Защита необходима, так как информация имеет ценность (пароли, банковские данные, корпоративные секреты). Без ИБ возможны утечки данных и финансовые потери.


**Основные цели ИБ (Триада)**

- **Конфиденциальность:** Информация доступна только тем, кому разрешено (например, пароль знает только владелец).

- **Целостность:** Данные нельзя изменить незаметно (например, банковский перевод не должен измениться по пути).

- **Доступность:** Информация и системы доступны тогда, когда они нужны (например, сайт банка работает 24/7).

**Основы шифрования**

- **Шифрование** — процесс преобразования информации в вид, нечитаемый без специального ключа.

- **Базовые понятия:** Открытый текст (исходник) превращается в Шифртекст с помощью Математического алгоритма и Секретного ключа.

**Простые методы шифрования:**

1. **Шифр Цезаря:** Каждая буква смещается по алфавиту на фиксированное число позиций.

2. **Шифр подстановки:** Каждой букве жестко соответствует другая случайная буква.

3. **Шифр Виженера:** Использует ключевое слово, где каждая буква шифруется своим, отдельным сдвигом.
