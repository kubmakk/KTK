# Учебные проекты (КТК)

Этот репозиторий представляет собой сборник практических и контрольных работ, которые я выполняю во время учебных пар в колледже. Здесь хранятся исходные коды, задания и заметки по различным дисциплинам, связанным с программированием.

## Содержание репозитория

### 1. Программирование на C# (Папка `чибисова`)

В этом разделе собраны работы по изучению объектно-ориентированного программирования (ООП) на языке C#

### 2. Программирование на C (Папка `соколов`)

Работы, посвященные основам алгоритмизации и изучению языка Си:

### 3. Геймдизайн и анализ (Папка `Зорин`)

Документация и аналитические материалы по разработке игр

## О проекте

Все файлы являются учебными материалами. Основная цель — отработка навыков написания чистого кода, понимания алгоритмов и проектирования игровых систем в рамках образовательной программы КТК.
