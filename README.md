# Мои учебные работы (КТК)

В этом репозитории собраны все практические задания, лабораторные и проекты, которые я выполняю на учебных парах в колледже.

### О проекте

Это не какой-то коммерческий продукт или завершенное приложение. Здесь просто хранится мой прогресс:

* Решение задач на **C#** (практические работы).
* Работа с базами данных (**SQL**).
* Настройки сетей и другие профильные задания.

Все файлы структурированы по предметам и фамилиям преподавателей для моего удобства.

**Зачем этот репозиторий?**

1. Чтобы иметь доступ к коду из любой аудитории колледжа.
2. Чтобы отслеживать свой прогресс в обучении.
3. Чтобы не потерять выполненные работы перед сессией.
